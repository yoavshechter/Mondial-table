# טבלת שיבוטים — מונדיאל 2026

דף ווב שמושך את טבלת הדירוג של קבוצת "שיבוטים" מ-365scores בזמן אמת,
מוסיף את ה"תוספת" הקבועה לכל שחקן, וממיין לפי הסכום.

כל מי שיש לו את הלינק יכול לפתוח ולראות טבלה מעודכנת. אין צורך להתקין כלום.

## מבנה הפרויקט

```
mondial-table/
├─ index.html        ← הדף שכולם רואים
├─ api/
│  └─ table.js       ← פונקציית שרת שמושכת מ-365 ומחברת תוספות
├─ vercel.json       ← הגדרות
└─ README.md         ← הקובץ הזה
```

## איך מעלים ל-Vercel (פעם אחת, ~5 דקות)

יש שתי דרכים. הדרך דרך האתר היא הכי פשוטה ולא דורשת התקנות.

### דרך א' — דרך אתר Vercel (מומלץ, בלי טרמינל)

1. היכנס ל-https://vercel.com והירשם (אפשר עם חשבון GitHub / Google — חינם).
2. צריך שהקבצים יהיו ב-GitHub. אם אין לך — צור ריפו חדש ב-https://github.com/new,
   והעלה אליו את כל הקבצים מהתיקייה הזו (כפתור "uploading an existing file").
3. ב-Vercel לחץ **Add New → Project**, בחר את הריפו, ולחץ **Deploy**.
4. אחרי דקה תקבל לינק קבוע כמו `https://mondial-table.vercel.app` — זה הלינק לשתף בקבוצה.

### דרך ב' — דרך הטרמינל (אם נוח לך)

```bash
npm i -g vercel       # התקנה חד-פעמית
cd mondial-table
vercel                # עוקב אחרי ההוראות, מתחבר, ומפרסם
vercel --prod         # פרסום לכתובת הקבועה
```

## איך מעדכנים תוספות

פתח את `api/table.js`. למעלה יש את הרשימה `ADDITIONS`.
כל שורה היא שחקן, לפי ה-userID הקבוע שלו ב-365:

```js
216719: { name: "אלדר", add: 7 },
```

- כדי **לשנות תוספת** — שנה את המספר ב-`add`.
- כדי **לשנות שם תצוגה** — שנה את הטקסט ב-`name`.
- כדי **להוסיף שחקן חדש** — הוסף שורה עם ה-userID שלו (יופיע בדף עם "(חדש!)" עד שתוסיף אותו).

אחרי שינוי: אם השתמשת בדרך א', ערוך את הקובץ ב-GitHub ו-Vercel יפרסם אוטומטית.
אם בדרך ב', הרץ שוב `vercel --prod`.

## הערה על הבוט "הקוף"

ה-userID של "הקוף" הוא 1, והוא מסומן להתעלמות (הוא בוט ולא משתתף אמיתי).
אם תרצה שכן יופיע — הסר אותו מ-`IGNORE_IDS` בקובץ `api/table.js`.

## נבחרת זוכה ומלך שערים (12 נק' כל אחד)

לכל שחקן יש שני ניחושים נוספים — נבחרת זוכה ומלך שערים — ששווים 12 נק' כל אחד,
ומתווספים רק בסוף הטורניר. הניחושים כבר מוגדרים בקובץ `api/table.js` ברשימת `PREDICTIONS`.

כל עוד הטורניר לא נגמר, שני המשתנים האלה בראש הקובץ נשארים `null`:

```js
const ACTUAL_WINNER = null;
const ACTUAL_SCORER = null;
```

ובמצב הזה כל הבונוסים הם 0, והדף מציג רק את הדירוג הנוכחי.

**בסוף הטורניר**, ערוך את שני המשתנים לתוצאה האמיתית — בדיוק כפי שהשמות כתובים
בניחושים (למשל "צרפת", "אמבפה"):

```js
const ACTUAL_WINNER = "צרפת";
const ACTUAL_SCORER = "אמבפה";
```

ברגע שתשמור ותפרסם מחדש, כל מי שניחש נכון יקבל אוטומטית 12 נק' לכל ניחוש,
והטבלה תמוין מחדש. בדף יופיע סימון ירוק ליד כל ניחוש מוצלח.

> שמות הנבחרות/שחקנים חייבים להיות **זהים בדיוק** בין הניחוש לבין התוצאה
> (כולל רווחים ואיות), אחרת הבונוס לא יזוהה.
